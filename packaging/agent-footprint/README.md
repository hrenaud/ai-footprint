# agent-footprint

Ce paquet redirige vers **[ai-footprint](https://pypi.org/project/ai-footprint/)**,
le nom retenu pour ce projet. Il n'installe aucun code propre : il déclare
`ai-footprint` en dépendance, afin que `pip install agent-footprint` continue de
fonctionner pour qui cherchait ce nom.

```bash
pip install agent-footprint   # installe en réalité ai-footprint
```

Utilise directement [`pip install ai-footprint`](https://pypi.org/project/ai-footprint/)
si tu démarres un nouveau projet.
